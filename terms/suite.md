# Suite
**Англійською:** suite  
**Українською:** випуск (suite)  
**Визначення:**  
У Debian термін *suite* означає логічний набір пакунків, який відповідає певному стану дистрибутива: `stable`, `testing`, `unstable`, `oldstable`, `experimental`.  
Suite визначає, які версії пакунків доступні користувачу.

**Примітки:**  
- Suite — це не гілка Git і не реліз у класичному сенсі.  
- Найважливіші: `stable`, `testing`, `unstable`.
