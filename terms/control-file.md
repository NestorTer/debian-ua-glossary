# Файл control
**Англійською:** control file  
**Українською:** файл control  
**Визначення:**  
Основний метаданний файл у каталозі `debian/`, який описує пакунок: залежності, опис, архітектуру, супровідника та інші параметри.

**Примітки:**  
- Формат визначений у Debian Policy.  
- Містить секції `Source:` і `Package:`.
