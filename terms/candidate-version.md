# Candidate Version
**Англійською:** candidate version  
**Українською:** кандидатна версія  
**Визначення:**  
Версія пакунка, яку APT вважає найкращою для встановлення або оновлення, з урахуванням пріоритетів, pinned‑правил і доступних репозиторіїв.

**Примітки:**  
- Відображається у `apt-cache policy`.  
- Може відрізнятися від найновішої доступної версії.  
- Визначається алгоритмом вибору APT.
