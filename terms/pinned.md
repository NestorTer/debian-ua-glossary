# Pinned
**Англійською:** pinned  
**Українською:** пришпилений  
**Визначення:**  
Стан пакунка або версії в APT, коли для нього вручну встановлено пріоритет у файлі `apt_preferences`, що впливає на вибір версії під час встановлення чи оновлення.

**Примітки:**  
- Використовується для контролю версій між `stable`, `testing`, `unstable`, backports.  
- Діє через механізм pin‑пріоритетів (Pin-Priority).  
- Діє як “фіксація” бажаної версії.
