# Mount Namespace Lock
**Англійською:** mount namespace lock  
**Українською:** блокування простору імен монтування  
**Визначення:**  
Ситуація, коли процес утримує власний простір імен монтування, що заважає відмонтувати файлову систему або каталог.

**Примітки:**  
- Часто виникає через фонові процеси, systemd‑юнити або контейнерні середовища.
