# Held Package
**Англійською:** held package  
**Українською:** утримуваний пакунок  
**Визначення:**  
Пакунок, який позначено як такий, що не повинен оновлюватися автоматично.  
APT не оновлює його навіть якщо доступна новіша версія.

**Примітки:**  
- Керується через `apt-mark hold` та `apt-mark unhold`.  
- Використовується для запобігання небажаним оновленням.
