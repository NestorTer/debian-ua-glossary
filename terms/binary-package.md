# binary package

## Визначення (англ)

**Debian Policy Manual**  
A binary package is a package that can be installed on a system. It contains the files and metadata needed for installation, configuration, and operation. Binary packages are produced by building source packages.

**General UNIX terminology**  
A binary package is a compiled or otherwise ready‑to‑install software unit, as opposed to a source package which contains source code and build instructions.

## Визначення (укр)

**Загальне визначення**  
Бінарний пакунок — це пакунок, який можна безпосередньо встановити в систему. Він містить файли, метадані та службові скрипти, необхідні для встановлення, налаштування та роботи програмного забезпечення.

**Debian / APT**  
Бінарний пакунок — це результат збирання вихідного пакунка. Він представлений у вигляді `.deb` файлу та містить готові до використання файли (виконувані, конфігураційні, документацію тощо).

## Контекст у APT

APT працює саме з бінарними пакунками:

- завантажує їх зі сховищ,
- перевіряє залежності,
- встановлює та оновлює,
- видаляє або очищує,
- визначає їхній стан (installed, unpacked, half-configured, broken).

Бінарний пакунок може не містити виконуваних файлів — наприклад, документацію або шрифти — але все одно вважається бінарним, бо є готовим до встановлення.

## Можливі переклади

- **бінарний пакунок**
- двійковий пакунок (помилково)
- готовий пакунок (не технічний термін)

## Аргументи за/проти

### «бінарний пакунок»

**За:**

- усталений термін у Debian, Ubuntu, KDE, GNOME;
- відповідає міжнародній UNIX‑термінології;
- не вводить в оману щодо змісту пакунка;
- узгоджується з протиставленням *binary vs source*.

**Проти:**

- іншомовне походження, але це прийнятно для технічних термінів.

### «двійковий пакунок»

**Проти:**

- створює хибне враження, що всередині обов’язково є двійкові виконувані файли;
- суперечить реальності (документаційні пакунки теж binary packages);
- не відповідає міжнародній практиці.

## Обраний переклад

**бінарний пакунок**

## Примітки

- Не плутати з «двійковими файлами» у значенні binary
