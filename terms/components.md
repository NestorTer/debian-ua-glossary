# Components
**Англійською:** main, contrib, non-free  
**Українською:** компоненти архіву Debian  
**Визначення:**  
Компоненти визначають рівень відповідності пакунків принципам Debian (DFSG).  
- `main` — повністю вільні пакунки.  
- `contrib` — вільні пакунки, що залежать від невільних.  
- `non-free` — пакунки з невільними ліцензіями.

**Примітки:**  
- Компонент визначає, у якій частині архіву буде розміщено пакунок.
